## ðŸŽ¯ Main Prompt Template for Blog Posts

Use this prompt template with ChatGPT to generate SEO-optimized blog titles, descriptions, and excerpts for your affiliate blog posts.

### Basic Prompt

```
You are an expert SEO content writer specializing in affiliate marketing and product reviews. 

Create content for a blog post about: [PRODUCT/CATEGORY NAME]

Requirements:
- Target audience: [e.g., "homeowners looking for kitchen appliances" or "tech enthusiasts seeking gaming gear"]
- Tone: [e.g., "helpful, trustworthy, and conversational" or "professional and detailed"]
- Include affiliate links naturally (mention this but don't create actual links)

Generate:
1. **Blog Post Title** (60-70 characters, SEO-optimized, includes primary keyword)
   - Should be compelling and click-worthy
   - Format: "Best [Product] for [Use Case] in [Year] - [Benefit]"
   - Example: "Best Air Fryers for Small Kitchens in 2026 - Space-Saving & Efficient"

2. **Meta Description** (150-160 characters, includes call-to-action)
   - Should summarize the value proposition
   - Include primary keyword naturally
   - End with a benefit or question

3. **Blog Excerpt** (150-200 words)
   - Engaging opening that hooks the reader
   - Mentions key benefits or pain points addressed
   - Teases what readers will learn
   - Includes primary and secondary keywords naturally

4. **5-7 SEO Keywords** (mix of primary, long-tail, and related terms)
   - Primary keyword (main focus)
   - Long-tail keywords (3-5 words)
   - Related/alternative terms

5. **3-5 Subheadings** (H2 tags for the article structure)
   - Should cover main topics/sections
   - Include keywords naturally

Please format the output clearly with labels for each section.
```

## ðŸ“ Specific Use Cases

### 1. Product Review Post

```
You are an expert product reviewer writing for an affiliate blog. 

Create SEO-optimized content for a review of: [PRODUCT NAME]

Context:
- Category: [e.g., "Kitchen Appliances" / "Tech & Gaming" / "Health & Wellness"]
- Subcategory: [e.g., "Air Fryers" / "Gaming Mice" / "Fitness Trackers"]
- Target audience: [e.g., "budget-conscious consumers" / "professional gamers" / "fitness enthusiasts"]
- Commission rate: [e.g., "4.5%" / "2.5%"]

Generate:
1. **Title** (60-70 chars): "Best [Product] Review [Year] - [Key Benefit]"
2. **Meta Description** (150-160 chars): Include price range, key features, and value proposition
3. **Excerpt** (150-200 words): Hook + what makes this product stand out + what readers will learn
4. **Keywords**: Primary + 5-7 long-tail keywords
5. **Article Outline**: 5-7 H2 headings covering: Overview, Key Features, Pros & Cons, Who It's For, Alternatives, Final Verdict

Focus on: Honest assessment, specific use cases, comparison context, and clear value proposition.
```

### 2. "Best of" Roundup Post

```
You are an SEO content writer creating a "Best Products" roundup article.

Topic: "Best [Product Category] for [Use Case] in [Year]"

Context:
- Category: [e.g., "Home & Lifestyle" / "Tech & Gaming"]
- Number of products: [e.g., "Top 10" / "Top 5"]
- Target audience: [e.g., "first-time buyers" / "experienced users"]
- Budget range: [e.g., "under $100" / "premium options"]

Generate:
1. **Title** (60-70 chars): "Best [Category] for [Use Case] [Year] - Top [Number] Picks"
2. **Meta Description** (150-160 chars): Number of products + key criteria + benefit
3. **Excerpt** (150-200 words): Why this roundup matters + selection criteria + what's included
4. **Keywords**: Primary + comparison keywords + buyer's guide terms
5. **Article Structure**: Introduction, Selection Criteria, Product Reviews (for each), Comparison Table, Buying Guide, Final Recommendations

Include: Clear selection criteria, honest pros/cons, specific use cases, and helpful comparisons.
```

### 3. Buying Guide Post

```
You are a product expert writing a comprehensive buying guide.

Topic: "[Product Category] Buying Guide [Year] - How to Choose the Right [Product]"

Context:
- Category: [e.g., "Smart Home Devices" / "Fitness Equipment"]
- Target audience: [e.g., "beginners" / "upgraders"]
- Price range: [e.g., "budget to premium"]

Generate:
1. **Title** (60-70 chars): "Complete [Category] Buying Guide [Year] - Expert Tips"
2. **Meta Description** (150-160 chars): What readers will learn + key factors to consider
3. **Excerpt** (150-200 words): Why this guide is needed + what's covered + who it's for
4. **Keywords**: Buying guide terms + category keywords + decision-making terms
5. **Article Structure**: Introduction, Key Factors to Consider, Features Explained, Price Ranges, Top Recommendations, Common Mistakes to Avoid, FAQs

Focus on: Education, decision-making help, feature explanations, and practical advice.
```

### 4. Comparison Post ("X vs Y")

```
You are a product comparison expert writing a head-to-head review.

Topic: "[Product A] vs [Product B] - Which is Better? [Year] Comparison"

Context:
- Category: [e.g., "Kitchen Appliances" / "Tech Accessories"]
- Products being compared: [Product A] and [Product B]
- Target audience: [e.g., "people deciding between these two options"]

Generate:
1. **Title** (60-70 chars): "[Product A] vs [Product B] [Year] - Detailed Comparison"
2. **Meta Description** (150-160 chars): Both products + key differentiator + recommendation hint
3. **Excerpt** (150-200 words): Why compare these + what's different + who should choose what
4. **Keywords**: Both product names + comparison terms + "vs" keywords
5. **Article Structure**: Overview, Side-by-Side Comparison, Feature Breakdown, Price Comparison, Pros & Cons, Use Case Recommendations, Winner & Verdict

Focus on: Clear differences, specific use cases, honest assessment, and clear winner recommendation.
```

## ðŸ” SEO Best Practices to Mention in Prompts

Add these instructions to your prompts for better SEO:

```
SEO Requirements:
- Include primary keyword in title (preferably at the beginning)
- Use natural keyword variations (don't stuff keywords)
- Include year in title for freshness (2026)
- Meta description should be 150-160 characters
- Include a call-to-action in meta description
- Use power words: "Best", "Ultimate", "Complete", "Expert", "Proven"
- Include numbers when relevant: "Top 10", "5 Best", "2026 Guide"
- Make titles specific: Include use case, benefit, or target audience
```
